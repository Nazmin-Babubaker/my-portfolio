- React
- Tailwind CSS
- Lucide (icons)

## Getting started (local)
Prerequisites:
- Node.js (v14 or newer)
- npm or yarn

Clone the repo:
```bash
git clone https://github.com/Nazmin-Babubaker/my-portfolio.git
cd my-portfolio
```

Install dependencies:
```bash
npm install
# or
yarn
```

Start local dev server:
```bash
npm run dev
# or
yarn dev
```

Build for production:
```bash
npm run build
# or
yarn build
```

Preview production build locally:
```bash
npm run start
# or
yarn start
```
